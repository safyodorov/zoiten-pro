## UI Surface 1: Список кредитов (/credits)

**a. Открытие страницы**
- [ ] `/credits` открывается без ошибок (нет 500, нет "Unauthorized")
- [ ] Список показывает кредиты (Сбербанк + JetLend, всего 23 кредита)

**b. Колонки таблицы**
- [ ] Присутствуют все колонки: Орг / Кредитор / № КД / Сумма / Ставка / Срок / Дата выдачи / Остаток / Статус
- [ ] Термин «Кредитор» (НЕ «Банк») в заголовке и фильтрах

**c. Фильтры**
- [ ] Фильтр по организации работает (выбрать «Зойтен» → остаются только кредиты Зойтен)
- [ ] Фильтр по кредитору работает (выбрать «JetLend» → остаются только JetLend кредиты)
- [ ] Фильтр по статусу работает (активен / погашён)
- [ ] Сброс фильтров возвращает полный список

**d. Добавление кредита**
- [ ] Кнопка «Добавить кредит» открывает модалку создания
- [ ] В модалке есть поле выбора кредитора (Сбербанк / JetLend)
- [ ] Создание тестового кредита сохраняется и появляется в списке
- [ ] Удалить тестовый кредит (или пометить к удалению)

**e. Редактирование кредита**
- [ ] Открыть любой кредит на редактирование (иконка или клик)
- [ ] Изменить примечание → сохранить → изменение отражается в UI
- [ ] Редактирование строки графика (inline или в модалке) сохраняется

**f. Удаление кредита (soft delete)**
- [ ] Soft delete кредита через кнопку удаления
- [ ] Кредит пропадает из списка (deletedAt проставлен)

## UI Surface 2: Детальная карточка (/credits/[id])

**g. Summary cards**
- [ ] Клик по кредиту → `/credits/[id]` открывается
- [ ] Summary cards показывают: сумма кредита / погашено тела / уплачено процентов / текущий остаток / переплата / ставка
- [ ] Кредитор отображён (Сбербанк или JetLend) — НЕ «Банк»
- [ ] Текущий остаток = Сумма кредита − Σ тело всех платежей

**h. Таблица графика + line-chart**
- [ ] Таблица графика показывает строки: дата / тело / проценты / остаток
- [ ] Остаток в каждой строке уменьшается накопительно (вычисляемый)
- [ ] Line-chart рисует кривую остатка основного долга (убывающая)
- [ ] Для JetLend кредита с PDF — строки отображают дневные/еженедельные даты (не только первые числа месяца)

## UI Surface 3: Сводный горизонтальный график (/credits/schedule)

**i. Открытие и структура**
- [ ] `/credits/schedule` открывается без ошибок
- [ ] Левый блок (sticky при горизонтальном скролле): организация / кредитор / № КД / сумма / ставка / остаток
- [ ] Горизонтальный скролл по колонкам-периодам работает
- [ ] Левый блок НЕ скроллится — остаётся на месте

**j. Переключение периодов**
- [ ] Переключатель день/неделя/месяц работает
- [ ] При «Месяц» — колонки отображают месяц (напр. «Апр 2024», «Май 2024»...)
- [ ] При «Неделя» — колонки показывают недельные периоды (ISO-неделя или дата начала)
- [ ] При «День» — колонки показывают конкретные даты

**k. Структура строк**
- [ ] Каждый кредит = 2 строки: строка «Тело» + строка «Проценты»
- [ ] В ячейках периодов — соответствующие суммы (тело и проценты попавших в период платежей)
- [ ] Per-org подытоговые строки отображаются (Зойтен итого / Дрим Лайн итого / Пеликан итого / Сикрет Вэй итого)
- [ ] Строка «Итого» внизу таблицы видна

## Ключевая сверка: Сводный график vs Лист2

**l. Сверка контрольных сумм (КЛЮЧЕВОЙ ПУНКТ)**

При месячной разбивке, диапазон апрель 2024 — декабрь 2026:

| Орг | Ожидаемая Σ тела | Ожидаемая Σ процентов |
|-----|-----------------|----------------------|
| Зойтен | 74 280 379,24 ₽ | 18 596 079,98 ₽ |
| Дрим Лайн | 56 261 014,34 ₽ | 11 337 869,94 ₽ |
| Пеликан | 10 783 800,00 ₽ | 264 325,34 ₽ |
| Сикрет Вэй | 7 193 280,00 ₽ | 435 156,51 ₽ |
| **ИТОГО** | **148 518 473,58 ₽** | **30 633 431,77 ₽** |

*(Числа из `.planning/phases/21-credits/21-04-SEED-NOTES.md` §7 — Контрольные суммы из Лист2)*

- [ ] Per-org суммы в сводном графике совпадают с ожидаемыми (допуск ≤ 100 ₽)
- [ ] Строка «ИТОГО» совпадает с ожидаемой (допуск ≤ 200 ₽)

**Дополнительно (U-04):**
- [ ] Для каждого кредита: накопленный остаток (сумма − Σtело платежей) сходится к 0 или близко к текущему остатку из БД

## UI Surface 4: Настройки

**m. Lender CRUD в /admin/settings**
- [ ] Открыть `/admin/settings` → присутствует таб «Кредиторы» (НЕ «Банки»)
- [ ] В табе отображаются Сбербанк и JetLend
- [ ] Можно добавить нового кредитора (ввод + Enter или кнопка «+»)
- [ ] Можно переименовать кредитора (карандаш → изменить → Enter)
- [ ] Попытка удалить Сбербанк или JetLend — показывает ошибку «Нельзя удалить кредитора с кредитами»
- [ ] Можно удалить тестового кредитора (если добавляли на предыдущем шаге)
