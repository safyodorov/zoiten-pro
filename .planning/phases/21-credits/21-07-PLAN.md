    ```typescript
    export interface PeriodColumn { key: string; label: string }
    export interface LoanScheduleRow {

      companyName: string
      lenderName: string                          // U-03: имя кредитора (loan.lender.name)
      amount: number

      currentBalance: number

      interestByPeriod: Record<string, number>   // key -> Σ interest
    }
    export interface OrgGroup {
      companyName: string
      loans: LoanScheduleRow[]
      subtotalPrincipalByPeriod: Record<string, number>
      subtotalInterestByPeriod: Record<string, number>
    }
    export interface SummarySchedule {
      columns: PeriodColumn[]
      groups: OrgGroup[]
      grandTotalPrincipalByPeriod: Record<string, number>
      grandTotalInterestByPeriod: Record<string, number>
    }
    ```

    `loadSummarySchedule(granularity: LoanGranularity, from: Date, to: Date): Promise<SummarySchedule>`:
    1. `prisma.loan.findMany({ where: { deletedAt: null }, include: { company, lender, payments: { orderBy: { date: "asc" } } } })`.

    3. Для каждого Loan: для каждого payment в окне [from,to] — `k = bucketKey(payment.date, gran)`; накапливать `principalByPeriod[k] += Number(principal)`, `interestByPeriod[k] += Number(interest)`. currentBalance = computeLoanAggregates(amount, allPayments).currentBalance (по ВСЕМ платежам, не только в окне). lenderName = loan.lender.name.
    4. Группировать loans по companyName (порядок орг: ПЕЛИКАН ХЭППИ ТОЙС, ЗОЙТЕН, СИКРЕТ ВЭЙ, ДРИМ ЛАЙН, затем прочие алфавитно). Внутри группы сортировать по lenderName → contractNumber.
    5. Подытоги per-org: суммировать principalByPeriod/interestByPeriod всех loans группы per period key.
    6. Grand total: суммировать по всем группам per period key.
    7. round2 на все суммы.
