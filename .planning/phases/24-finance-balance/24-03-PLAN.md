WB API — ОФИЦИАЛЬНО ВЕРИФИЦИРОВАНО (RESEARCH §«Официальная документация», HIGH):
- Balance: `GET finance-api.wildberries.ru/api/v1/account/balance` → ответ
  `{"currency":"RUB","current":10196.21,"for_withdraw":6395.8}` — ЧИСЛА (не строки).
  Rate limit: Персональный/Сервисный = 1 req/мин; Базовый = 1 req/24ч.
  Ошибки: 401 / **402 Payment Required** («Требуется платёж» — обрабатывать явно) / 429.
- Sales tail: `GET statistics-api.wildberries.ru/api/v1/supplier/sales?dateFrom=YYYY-MM-DD&flag=0` → поле `forPay`.
  ⚠ Токен — Статистика (WB_API_TOKEN), НЕ Финансы (B4). ⚠ flag=0 → dateFrom = фильтр lastChangeDate,
  не дата продажи → пост-фильтр по saleDt обязателен (M1).
- ⚠ НЕ использовать `api/v5/supplier/reportDetailByPeriod` (официально deprecated, удаляется 15.07.2026).
- Будущий слой (deferred, НЕ в v1): `finance/v1/sales-reports/*` — деньги СТРОКИ, `paymentSchedule` строка,
  daily reportId требует BigInt. Оставить комментарий в шапке клиента для будущей фазы.
