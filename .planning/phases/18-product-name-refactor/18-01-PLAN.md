## Этап 2: Pure generation function

**Файл:** `lib/product-name.ts` (новый)

```typescript
export interface ProductNameInput {
  article: string
  category?: { name: string } | null
  subcategory?: { name: string } | null
  brand?: { direction?: { hasSizes: boolean } | null } | null
  // propertyValues с дополнительной инфой про includeInName флаг
  properties?: Array<{ value: string; includeInName: boolean }>
}

export function generateProductName(input: ProductNameInput): string {
  const hasSizes = input.brand?.direction?.hasSizes === true
  const parts: (string | null | undefined)[] = hasSizes
    ? [
        input.category?.name,
        input.subcategory?.name,
        ...(input.properties ?? []).filter((p) => p.includeInName).map((p) => p.value),
        input.article,
      ]
    : [
        input.subcategory?.name ?? input.category?.name,
        input.article,
      ]
  return parts
    .map((p) => (p ?? "").trim())
    .filter((s) => s.length > 0)
    .join(" ")
}
```

Pure, детерминированная, без I/O. Используется и на сервере, и потенциально на клиенте (для preview в форме).

## Этап 3: Server-side regenerate helper

**Файл:** `app/actions/products.ts` (добавление)

```typescript
async function regenerateProductName(tx: PrismaTx, productId: string): Promise<void> {
  const p = await tx.product.findUnique({
    where: { id: productId },
    include: {
      brand: { include: { direction: true } },
      category: true,
      subcategory: true,
      propertyValues: { include: { property: true } },
    },
  })
  if (!p || p.nameOverridden) return
  const name = generateProductName({
    article: p.article,
    category: p.category,
    subcategory: p.subcategory,
    brand: p.brand,
    properties: p.propertyValues.map((pv) => ({
      value: pv.value,
      includeInName: pv.property.includeInName,
    })),
  })
  await tx.product.update({ where: { id: productId }, data: { name } })
}
```

Вызовы:
- `createProduct` (после nested create) → `regenerateProductName(tx, product.id)`
- `updateProduct` (после update + replace-all marketplaces) → `regenerateProductName(tx, productId)`
- `saveProductProperties` (после upserts) → `regenerateProductName(tx, productId)`
- `importFromWb` (после import — там уже есть Product.update updatedAt) → `regenerateProductName(tx, productId)`
