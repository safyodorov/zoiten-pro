<api_facts>
Живой Analytics warehouse_remains (база `https://seller-analytics-api.wildberries.ru`, токен WB_API_TOKEN через getToken):
1. CREATE:  `GET /api/v1/warehouse_remains?groupByNm=true&groupBySize=true&groupByBarcode=true` → 200 `{"data":{"taskId":"..."}}`
2. STATUS:  `GET /api/v1/warehouse_remains/tasks/{taskId}/status` → `{"data":{"id":"...","status":"done"}}` (готово ~20 сек)
3. DOWNLOAD:`GET /api/v1/warehouse_remains/tasks/{taskId}/download` → JSON-массив строк:
```json
[{ "nmId": 348382132, "barcode": "2043093896721", "techSize": "0", "volume": 2.88,
   "warehouses": [
     {"warehouseName": "В пути до получателей", "quantity": 42},

     {"warehouseName": "Всего находится на складах", "quantity": 1491},
     {"warehouseName": "Коледино", "quantity": 1}
   ]}]
```
Rate limit: создание задания ~1 req/мин. Поллинг статуса — отдельный лимит.
